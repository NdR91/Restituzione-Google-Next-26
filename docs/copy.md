Workspace

## L’agente entra nel lavoro quotidiano.

Workspace diventa il contesto operativo degli agenti. Ogni strumento — dalla Chat al Canvas, dalle Skills all'MCP — è progettato per far lavorare l'AI dentro i processi quotidiani.

### Ask Gemini in Chat

Daily briefing, task urgenti e comandi conversazionali collegati a Workspace e tool terzi.

### Canvas Mode

Creazione e modifica di documenti e slide, con export Office e contesto condiviso.

### MCP Server

Capability Workspace disponibili dentro applicazioni AI esterne, con framework aperto e sicuro.

### M365 migration

Nuovo servizio cloud-based: migrazioni fino a 5x più rapide e gancio competitivo diretto.

Sales playbook

## Una mappa per aprire conversazioni.

COO / Operations

### Forza lavoro digitale

Agent Designer, long-running agents, Skills: processi end-to-end, non chatbot.

CIO / CTO

### Agent factory

Agent Platform, runtime, ADK, observability: industrializzare sviluppo e rilascio.

CISO

### Governance by design

Identity, Registry, Gateway, Wiz, Model Armor: sicurezza come prerequisito enterprise.

CDO / Data

### Prima i dati

Knowledge Catalog, BigQuery, semantic layer: senza fondazione dati, gli agenti non scalano.
